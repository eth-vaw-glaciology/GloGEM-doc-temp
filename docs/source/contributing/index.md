# Contributing

GloGEM is developed and maintained by the VAW glaciology group at ETH Zürich. Both the model code and this documentation are hosted on GitHub and accept contributions from group members.

There are two separate repositories:

| Repository | URL | Purpose |
|------------|-----|---------|
| GloGEM (code) | https://github.com/eth-vaw-glaciology/GloGEM | IDL model code |
| GloGEM-doc-temp (docs) | https://github.com/eth-vaw-glaciology/GloGEM-doc-temp | This documentation |

Both repos use a shared `main` branch. All contributors push directly to `main`, so coordinating changes and keeping your local copy up to date is important to avoid conflicts.
